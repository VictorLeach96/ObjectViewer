# Object Viewer
OpenGL wavefront .obj loader written in C++.

## Features

  * Loads Triangulated objects
  * Texturing (BMP, PNG, TGA...)
  * Material Groups & Properties
  * Basic Lighting
  * Normal & Specular Mapping

## Libraries

  * OpenGL
  * GLM
  * GLEW
  * SDL2
  * SDL2 Image

## Controls

  * Q - Zoom In
  * E - Zoom Out
  * W - Rotate Up
  * A - Rotate Left
  * S - Rotate Down
  * D - Rotate Right
